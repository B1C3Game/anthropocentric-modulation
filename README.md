# B1C3 Clock

B1C3 Clock is an anthropocentric time model that represents time in human-meaningful rhythms rather than conventional 12/60 clock divisions. It frames each day as six 4-hour bouts, includes a recovery cadence inside each bout, and adds a 7-phase fermentation metaphor for multi-day progression.
